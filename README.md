# Go语法快速学习指南

这个项目包含了Go语言的核心语法概念，通过实例代码帮助你快速掌握Go语言。

## 学习顺序
1. **基础语法** - `01_basics/`
2. **数据类型** - `02_types/`
3. **控制流** - `03_control_flow/`
4. **函数** - `04_functions/`
5. **结构体与方法** - `05_structs_methods/`
6. **接口** - `06_interfaces/`
7. **并发编程** - `07_concurrency/`
8. **包管理** - `08_packages/`
9. **错误处理** - `09_error_handling/`
10. **实战练习** - `10_practice/`

## 如何使用
1. 按顺序学习每个目录中的内容
2. 运行代码示例：`go run filename.go`
3. 完成每个部分的练习
4. 查看解答和最佳实践

## 快速开始
```bash
cd 01_basics
go run hello.go
```

祝你学习愉快！🚀
